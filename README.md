## Features

With this code, all ghosts have a red circle from which you can see their angle of view.
(ghost.js , draw())

```javascript
 canvasContext.strokeStyle = "red";
         canvasContext.arc(
             this.x + oneBlockSize / 2,
             this.y + oneBlockSize / 2,
             this.range * oneBlockSize,
             0,
             2 * Math.PI
         );
```
